# read more

**「続きは、続きを読むからどうぞ！」**——友人（zaftx / 4g.MiNaMi / さかやん）が毎回ブログに書いていた、あの定型文だけで作られたばかゲー。

👉 **遊ぶ: https://d-shiina.github.io/read-more/**

「続きを読む」を押すと、続きが**その場の下に生えてくる**。スライドしない。記事が積み上がる。
2011年の記事から始まって、続きを読み続けると日付が15年進む——「“続き”に15年かかった」という彼のブログの実話が、そのまま構造になっている。

## 特徴

- **全17ステージ、1ステージ＝1ボケ**（再帰／オフ会カラオケ／偽ローディング／逃げるボタン／タブ汚染／黒歴史Twitter／スクロール地獄／∞／増殖／2011年デザイン／偽最終回／実在の「お前らしばくぞｗ」…）
- **編注（2026年の本人）**：黒歴史Twitterステージ限定の、未来の自分によるセルフツッコミ（1ツイートごと＋集計つき）
- **実績20個**：素直に進めるだけでは5個前後しか取れない。撫でる・連打する・掘る・∞に挑む・召喚する…寄り道した者だけが集められる。**排他ペア**（追うか待つか／狙撃か被弾か）があるので1周でコンプ不可
- **隠し実績**：コナミコマンド↑↑↓↓←→←→BA ／ 開発者コンソール（F12して言われた通りにする）
- 効果音はWebAudio生成（アセットゼロ）、進行はlocalStorageにセーブ

## 元ネタ

`blog-backup/README.md` に彼のブログ全10記事＋2012年の実在ツイート（匿名化済み）。ネタは全部ここから。

## 技術

Next.js 16（App Router）+ React 19 + TypeScript + Tailwind v4。
実装は `src/game/ReadMore.tsx` の1ファイル。GitHub Pages に静的書き出しで自動デプロイ（`.github/workflows/deploy-pages.yml`）。

```bash
npm install
npm run dev   # http://localhost:3000
npm run build
```
