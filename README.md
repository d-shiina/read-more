# read more

**「続きは、続きを読むからどうぞ！」**——友人（zaftx / 4g.MiNaMi / さかやん）が毎回ブログに書いていた、あの定型文だけで作られたばかゲー。

「続きを読む」を押すと、続きが出る。……と思うじゃん？

## しくみ

全16ステージ。**1ステージ＝1つのボケ**で、毎回ちがう手口で裏切ってくる。

|---|---|---|

記事の日付が 2011→2026 に勝手に進んでいく＝**“続き”に15年かかった**という、彼のブログの実話がそのまま構造になっている。

- ステージ通過で**実績解除**（全16個・効果音つき）
- 進行は localStorage にセーブ（途中でやめても続きから）
- 効果音は WebAudio 生成（アセット不要）、🔊/🔇切替あり
- 攻略はありません

## 技術

Next.js 16（App Router）+ React 19 + TypeScript + Tailwind v4。
実装は `src/game/ReadMore.tsx` の1ファイル（ステージ＝コンポーネント）。

```bash
npm install
npm run dev   # http://localhost:3000
npm run build
```
