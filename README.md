記事の日付が 2011→2026 に勝手に進んでいく＝**“続き”に15年かかった**という、彼のブログの実話がそのまま構造になっている。

- ステージ通過で**実績解除**（全18個・効果音つき）
- 進行は localStorage にセーブ（途中でやめても続きから）
- 効果音は WebAudio 生成（アセット不要）、🔊/🔇切替あり
- 攻略はありません

### 隠し要素（言わなきゃ気づかない系）

- **コナミコマンド**（↑↑↓↓←→←→BA）をどのステージからでも入力すると隠し実績
- **開発者コンソール**を開くとメッセージが出る。指示通り `tsudzuki()` と打つと隠し実績
- 「誠意を見せて」ステージで `つづき` の代わりに `Joka` と入力すると特別な返事

## 元ネタ・プライバシー方針

`blog-backup/README.md` に彼のブログ全10記事＋当時のツイート抜粋のバックアップ。ネタは全部ここから。
本人（zaftx）以外の第三者（返信相手・友人）の実アカウント名は、ゲーム内・資料内どちらからも伏せてある。

## 技術

Next.js 16（App Router）+ React 19 + TypeScript + Tailwind v4。静的書き出し（`output: "export"`）で GitHub Pages にホスト。
実装は `src/game/ReadMore.tsx` の1ファイル（ステージ＝コンポーネント）。

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # out/ に静的書き出し
```

main への push で `.github/workflows/deploy-pages.yml` が自動デプロイする。
