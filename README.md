## 技術

Next.js 16（App Router）+ React 19 + TypeScript + Tailwind v4。
実装は `src/game/escape/`（シェル／ブログビューア／DM／章パズル／エンディングに分割）。
GitHub Pages に静的書き出しで自動デプロイ（`.github/workflows/deploy-pages.yml`）。

```bash
npm install
npm run dev   # http://localhost:3000
npm run build
```
