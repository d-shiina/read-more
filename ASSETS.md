# アセット差し替えガイド（仮画像 → 本物のイラスト）

いまゲームに入っているのは **仮の SVG プレースホルダ**。
下の一覧のファイルを、生成した本物の画像（PNG / JPG / WebP）に置き換えるだけで絵が変わります。

## 差し替え手順

1. 画像を生成する（下の「生成プロンプト」を使う）
2. `public/images/...` に、**同じファイル名で拡張子だけ変えて**置く
   （例：`room-night.svg` → `room-night.png`）
3. `src/game/assets.ts` のパスを新しい拡張子に変える
   （例：`room: "/images/scenes/room-night.png"`）

> ⚠️ 背景（room-night）の**構図を大きく変えたら**、`src/game/ui/Scene.tsx` の
> ホットスポットの `x / y / w / h`（％座標）も合わせて調整してください。
> モニター・窓・ポスター・引き出し・ベッド・ドアの位置に対応しています。

## 必要な画像一覧

| ファイル | 用途 | 推奨サイズ / 比率 |
|---|---|---|
| `public/images/title-bg.svg` | タイトル背景 | 1280×800（16:10） |
| `public/images/scenes/room-night.svg` | 背景：深夜の自室 | 1280×800（16:10） |
| `public/images/chars/kona.svg` | こなちゃんの立ち絵／アイコン | 正方形 512×512 以上（背景透過だと理想） |
| `public/images/items/usb.svg` | アイテム：USBメモリ | 正方形 256×256（透過PNG推奨） |
| `public/images/items/disc.svg` | アイテム：焼いたDVD-R | 正方形 256×256（透過PNG推奨） |
| `public/images/puzzles/frag-card.svg` | 謎のカード | 800×560 前後（4:3〜16:10） |

## 生成プロンプト集（Google Gemini「Nano Banana」想定 / Midjourney でも可）

### 0. まず“スタイルシート”を1枚作る（超重要）

各アセットをバラバラに作ると絵柄が揃わない。**最初に基準の1枚**を作り、
以降は「この画像と同じ画風・同じキャラで」と参照させて量産する。

```
深夜のゲーマーの自室を舞台にした、ノスタルジックな日本のアニメ調イラスト。
2000年代後半のネトゲ文化の空気感。ダークで少し寂しげ、でも温かい。
配色は 深い紫紺 + ネオンの水色(#4be3ff)と紫(#b46bff) + 月明かりの暖色。
やわらかい塗り、うっすらグロー、フィルムグレイン少々。文字は入れないで。
```

### 1. 背景：深夜の自室（room-night）

```
【スタイルシートと同じ画風で】
深夜のワンルーム、ゲーマーの部屋を正面やや引きの構図で。横長16:10。
・左に大きな窓：外は夜、満月と遠くの街の灯り、少し開いたカーテン
・中央に机とゲーミングモニター（画面が青白く光る）、ゲーミングチェアの背、RGBの帯
・右壁にゲームのポスター、右手前にドア
・左手前にベッド、床にはエナジードリンクの空き缶やカップ麺
・机の右側に引き出し
キャラクターや文字は入れない。クリックする各家具がはっきり見える構図で。
```

### 2. こなちゃん（kona）

```
【スタイルシートと同じ画風で】
「こなちゃん」——ネトゲで一緒に遊んでいた女の子。ヘッドセットを着けた
やさしい雰囲気の少女。上半身の立ち絵、背景は透過（または単色）。
表情ちがいで数枚（にっこり／照れ／いたずらっぽく笑う）あると会話に使える。
※実在の人物には寄せず、あくまで“それっぽい”オリジナルキャラで。
```

### 3. アイテム：USBメモリ（usb）／ DVD-R（disc）

```
【スタイルシートと同じ画風で】アイコン用、背景透過の正方形。
・USBメモリ：紫のボディに「革命」と手書き風の文字。斜めから。
・DVD-R：手書きで「完成版・最終・本当の最終」と書かれた焼きディスク。
どちらもゲームの持ち物アイコンとして映える、少しデフォルメした一つ。
```

### 4. 謎のカード（frag-card）

```
【スタイルシートと同じ画風で】横長のカード。
FPSの“フラグムービー”をモチーフにしたフィルムストリップ。
コマの中に「A」「e」と、伏せ字の「？」「？」。
下に日本語で「これは革命だ、と叫んだ動画編集ソフトの名前は？」。
レトロで少しエモい雰囲気。
```

## 実在ネタの差し替え

謎の答えは `src/game/config.ts` の `ANSWERS` に集約。
いまは仮で `aftereffects`。友人のハンドル・こなちゃん関連の言葉・思い出の日付などに変える。
会話文は各シーン（`src/game/ui/Scene.tsx`）とパズル（`PuzzleModal.tsx`）内にある。
