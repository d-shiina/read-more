# read more: ESCAPE — ゲーム設計書 v1

> 「続きを読む」ばかゲー（v2.1）を、**ブログそのものが脱出部屋になる脱出ゲーム**にリプレースする。
> 元ネタは全て `blog-backup/README.md`（彼のブログ全10記事＋2012年のこなちゃんツイート14本）。

## 1. コンセプト

**「15年放置されたブログの中に閉じ込められた。管理人の"未練"を全部解かないと、出られない。」**

- 部屋＝ブログ。**ゲーム画面の中央に本物のブログ表示エリア**があり、彼の実記事がそのまま読める。謎の答えは全部、記事の中にある。
- 脱出の扉＝**「続きを読む」ボタン**。15年間誰も押せなかった"本当の続き"を読むことが、脱出条件。
- 並行して**こなちゃん好感度（こな度）**が存在し、寄り道の積み重ねで最終章の分岐が変わる。**ハッピーエンドは実在する。**

### 旧作からの継承DNA
- 実績システム（寄り道した者だけが集められる／排他ペアあり）
- WebAudio効果音（アセットゼロ）／localStorageセーブ
- 隠し要素（コナミコマンド／開発者コンソール）
- 「編注（2026年の本人）」のセルフツッコミ芸

## 4. ゲームフロー — 5つの未練（章構成）

各章 = 「錠前（ロック）」1つ。答えは解錠済みの記事の中に必ず書いてある。ノーヒントで解ける難度にしつつ、残留思念が段階ヒント（3段階）をくれる。

### 序章「凍結通知」
- チュートリアル。最初の記事（2011-04-16）を読む → プロフィール欄の**Skypeアイコン**をタップ → ロック入力欄が開放。
- 最初のロック: 「僕に連絡したい人はSkypeで**___**まで」→ 答え `zaftzaft3`（記事に原文ママ）。
- 解錠報酬: アーカイブ「2011年7月」解放＋こなからのDM開通。

### 第1章 未練①「試験」— 開かずの試験会場
- 対象記事: 「基本情報技術者試験（FE）」(2011-07-12)
- 謎: 4桁PINロック。ヒント「『あ～ｗ詰んだくせぇ・・・』が生まれた日」→ 答え `0710`（7月10日受験）。
- 追加ギミック: 選択式ミニ試験（4問、全部彼のブログが出典。例:「彼のマウスパッドは？」→ ARTISAN飛燕）。**極端に合ってる設問と極端に間違ってる設問しか出ない**（原文再現）。
- 報酬: アイテム「受験票の半券」／アーカイブ2011年7月後半解放。
- こな度チャンス: DMでこなに「試験どうだった？」と聞かれる → 「秋にリベンジする」を選ぶと+1（「不合格だったｗ」でも減りはしない。素直さが正義）。

### 第2章 未練②「クラン」— 4gottenの意味
- 対象記事: 「オフ会してきた！」(2011-07-18)、「ブログの存在を忘れかけていたぜ」(2011-08-02)
- 謎A: **カラオケ採点表パズル**。5人（楓/きいたりもん/フィッツ/hellfox/私）と特徴（DeathVoice担当/ハッチポッチステーション/控えめ/加藤ミリヤのGong/・・・///）のマッチング。
- 謎B: 全員正解すると採点表に「このクラン名、英単語にすると？」→ 答え `forgotten`（4gotten = forgotten。忘れられたブログ、という全体テーマの回収）。
- 報酬: アイテム「クランタグ 4g.」／アーカイブ2012年3月解放。
- こな度チャンス: 「私」の選曲だけ「・・・///」で隠されている。DMでこなが「あの人何歌ったか知ってる？」→ここで**カラオケの機種の選曲履歴を掘る寄り道**をすると隠しツイート発掘＋こな度+1。

### 第3章 未練③「師匠」— スナイパーの遺言
- 対象記事: 「CS:Sのクランお誘い」(2011-07-14)、「TOP」(2013-12-25)
- 謎: **誤字スナイプ**。スコープUI（マウス/タッチ追従の照準）で、記事の中の誤字を撃ち抜く。標的は実在の誤記 **「Alliance of Variant Arms」**（正: Valiant）。ヘッドショット判定あり。
- 撃ち抜くとJokaの残したメモが出る:「エイムは、ずらして合わせろ。——最終問題のヒントだ」
- 報酬: アイテム「SRの弾丸」／アーカイブ2013年解放。
- 実績（排他ペア）: 一発で誤字を狙撃 🎯「Jokaの後継者」／3回以上外す 🪖「グレで飛びすぎ」
- こな度チャンス: 検索欄に「マイクラ」と入れると2012-03-12のツイート発掘 →「鯖、立てておいたよ」をDMで送るとこな度+2（**14年越しの鯖立て**）。

### 第4章 未練④「革命」— 未製作のフラグムービー
- 対象記事: 「ブログ更新続けれるかな！？」(2011-07-12)＝AfterEffect革命の回
- 謎: **タイムライン編集**。これまで集めたアイテム（受験票の半券／クランタグ／SRの弾丸／＞ｗ＜）を、AfterEffect風タイムラインに**時系列順**に配置 → 正しい順（＝ブログの歴史順）でレンダリング開始。
- レンダリングは旧作の偽ローディングのオマージュ。98%で止まる……が、**今回は本当に100%になる**。15年越しに、フラグムービーが完成する（テキストアニメ約15秒、彼の15年をプレイバック）。
- 報酬: 鍵アイテム「革命」／アーカイブ2026年解放（「お前らしばくぞｗ」が読める）。
- こな度チャンス: ムービーのクレジットに「Special Thanks」欄が空いている。こなの名前を入れるか選べる（入れると+2。ただし**恥ずかしがって「みるくふぁいたー」と入れるとバッド寄りフラグ**）。

### 最終章 未練⑤「こなちゃん」— 黒歴史サルベージ
- 対象: 旧Twitterログ端末（2012年のツイート14本が読める）
- 演出: 端末の時が**2012年7月10日、あのダジャレツイートの送信直前**で止まっている。
  > 「こなだけに、粉まみれってか！」——【ツイートする】ボタンに指がかかったまま、14年。
- 最終謎: Jokaの遺言「エイムは、ずらして合わせろ」の回収。ツイート本文を**1文字だけずらして**本当に言いたかった言葉に直す入力パズル（答え例: `こな` → 「こなちゃん、今度九州の話きかせて」等、複数正解のガバ判定は旧作の伝統として維持）。
- ここで**最終選択**（こな度により選択肢が変化）:
  1. **そのまま送信する**（史実ルート）
  2. **送信せず、直した言葉を送る**（こな度4以上で出現）
  3. **何も送らず、会いに行く**（こな度MAX(6)＋隠し条件で出現）

### 脱出
最終選択後、ブログのTOPに**「続きを読む」ボタンが1つだけ**出現する。15年間存在しなかった本物。押すと脱出＝エピローグへ。

## 5. こな度とエンディング分岐

### こな度（0〜6）
| 稼ぎ方 | 値 |
|---|---|
| DMの選択肢で素直な返事を選ぶ（各章1回） | +1×4 |
| 14年越しのマイクラ鯖立て（第3章寄り道） | +2 |
| ムービーのSpecial Thanksにこなの名前（第4章） | +2 |
| ツイート14本を全部読む | +1 |
| デザートの差し入れ（隠し: 検索欄に「デザート」） | +1 |

※上限6でカンスト。UIは画面隅の顔文字メーターが `( ´・ω・)` → `(・ω・)` → `(＞ω＜)` と育つ（旧作「撫でると育つ」の継承）。

### エンディング3種＋隠し
| END | 条件 | 内容 |
|---|---|---|
| **C: 粉まみれエンド**（バッド） | 最終選択①（ダジャレ送信） | 既読がつく。返事はない。脱出はできるが、ブログの凍結通知だけが残る。「数字は嘘をつかない。」→ リトライ推奨の演出（実績「歴史は繰り返した」）。 |
| **B: 脱出エンド**（ノーマル） | 最終選択②、こな度4〜5 | 直した言葉が届き、「ありがとう」と返事が来る。脱出成功。エピローグで凍結が解除され、ブログに15年ぶりの新規記事が生える。 |
| **A: 寝落ちもしもしエンド**（ハッピー）【決定】 | 最終選択③、こな度6＋ツイート全読 | 送信ではなく通話発信（TSこいやぁああああの回収）。呼び出し音、そして——「……もしもし？」。23時から通話が続き、ニコ動・マイクラ鯖・デザートの話で夜が更け、深夜3時、返事が寝息に変わる。**「……おやすみ。切らないでおくね」**——寝落ちもしもし成立。エピローグ: 2026年の新規記事「【ご報告】寝落ちもしもしについて」が生える。最後の一文は**「続きは、二人で読むからどうぞ！＞ｗ＜」**。 |
| **隠し: Jokaエンド** | 最終ロックに「Joka」と入力し続ける | 師匠が降臨して代わりに全部解いてくれる（実績のみ、正史ではない）。 |

- **1周でA到達は不可能ではないが寄り道必須**（旧作の「素直に進めると実績5個」哲学の継承）。
- バッドエンド後は**こな度と実績を引き継いで2周目**が始まる（強くてニューゲーム）。

## 7. 技術設計

- **スタック**: 現行のまま Next.js 16 (App Router, 静的書き出し) + React 19 + TS + Tailwind v4。GitHub Pages デプロイも現行ワークフロー流用。
- **実装前の必須手順**: `AGENTS.md` の指示どおり `node_modules/next/dist/docs/` の該当ガイドを読んでから着手（Next.js 16 は破壊的変更あり）。
- **ファイル構成**（旧作の1ファイル1390行の反省を踏まえ分割）:

```
src/game/escape/
  EscapeGame.tsx        # シェル: レイアウト・セーブ・実績・効果音
  state.ts              # useReducer: chapter / solved / inventory / konaLevel / flags
  data/articles.ts      # blog-backup 由来の記事データ（原文）
  data/tweets.ts        # 2012年ツイート14本
  data/dm.ts            # こなちゃんDMスクリプト（章×選択肢）
  data/achievements.ts
  parts/BlogViewer.tsx  # ブログ表示エリア（livedoor風・アーカイブ錠前）
  parts/Sidebar.tsx     # プロフィール/残留思念/検索/カウンタ
  parts/DMWindow.tsx    # こなちゃんチャット＋こな度メーター
  parts/LockInput.tsx   # ロック入力（ガバ判定エンジン）
  parts/Inventory.tsx
  chapters/Ch0Intro.tsx / Ch1Exam.tsx / Ch2Karaoke.tsx /
  chapters/Ch3Snipe.tsx / Ch4Render.tsx / Ch5Kona.tsx / Endings.tsx
```

- **セーブ**: `localStorage` キー `readmore.escape.v1`。`{ chapter, solved[], inventory[], konaLevel, flags{}, ach[], startedAt, lap }`。旧キー `readmore.v4` は読まない（別ゲー扱い）が、存在したら実績「📖 前作読者」ボーナス…は次版検討。
- **効果音**: 旧作の `useSfx`（WebAudio生成）を移植。狙撃音・レンダ完了ファンファーレ・DM着信音を追加。
- **リプレース方針**: `src/app/page.tsx` を `EscapeGame` に差し替え。`ReadMore.tsx` は削除（git履歴に残る）。README・`layout.tsx` のメタデータも更新。セーブキーが別なので旧セーブと衝突しない。

## 9. 決定事項（2026-07-07 レビュー反映）

1. **旧ばかゲーは完全削除**。git履歴にのみ残る。検索欄に「続き」と入れると「旧作は完全に削除されました」の供養メッセージだけ出す。
2. **ブログ表示エリアは2026年のモダンすぎるデザイン**。レトロ再現ではなく「AIリマスターされた過剰モダンUI × 2011年の素朴な本文」のギャップで笑わせる。
3. **ハッピーエンド＝こなちゃんと寝落ちもしもし**（END A改稿済み、§5参照）。
4. バッドエンド（END C）は現行の温度感で維持し、強くてニューゲーム（こな度・実績引き継ぎ）でリトライ導線を用意する。
5. こなちゃんの性格付けはツイート逆算路線（ニコ動好き・デザートにうるさい・泣き顔文字期あり）で確定。
